Pyle 2

Pyle2 is a massive python library designed to simplify everyday tasks. It contains 50+ modules, and features some extensive documentation, making it very useful to use. It is also faster than most python libraries, due to some code being written in C.

Termicod

Termicod is a python module for printing text in any colour or style in the terminal. Termicod also supports progress bars as an extra feature, enabling you to create your own CLIs. Fonts and other things are (hopefully) going to be added in the future.

Docs: See PyPi.
Languages: Python 3
Version: 1.0.0
Download: PyPi

Unitrees

Unitrees is a very simple Python module for printing out little text "trees" generated from lists. These text trees can be viewed in any generic unicode-based terminal. They only work in fonts with fixed letter width. Also has ascii-compatible character sets.

Docs: See PyPi.
Languages: Python 3
Version: 1.0.0
Download: PyPi
