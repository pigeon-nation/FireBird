FireCloud

FireCloud is an online platform that contains many useful scripts and services, like online JavaScript and CSS resources, backend services for apps to communicate with, socket servers, generic embeddable pages and more!!

FireFHS

FireFHS is another online collection of documents, mainly hosted via GitHub as raw content. The FireFHS library allows your programs to access, read and import this online content with incredible ease.

DownNet

DownNet is a small, off to the side extension of FireCloud that mainly deals with making sure all our packages are up-to-date. It also has a few other functions which can prove useful when using our packages.
