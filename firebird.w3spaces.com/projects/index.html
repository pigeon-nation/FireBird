Pyle 2

Pyle2 is a massive python library designed to simplify everyday tasks. It contains 50+ modules, and features some extensive documentation, making it very useful to use. It is also faster than most python libraries, due to some code being written in C.

PyOnline

PyOnline is a combination of FireCloud and LibConnect. It is meant to allow python scripts to access online services, but instead of being limited to just the FireCloud services, it can read and data stored on the net. Not only does it deal with python code, but it can deal with ANY LibConnect-Language there is. Requires LibConnect.

LibConnect

LibConnect is a Python utility designed to help link python with a multitude of other programming languages through one, simple, easy interface. It currently supports JavaScript, C and C++.
