Termicod

Termicod is a python module for printing text in any colour or style in the terminal. Termicod also supports progress bars as an extra feature, enabling you to create your own CLIs. Fonts and other things are (hopefully) going to be added in the future.

Docs: See PyPi.
Languages: Python 3
Version: 1.0.0
Download: PyPi

Unitrees

Unitrees is a very simple Python module for printing out little text "trees" generated from lists. These text trees can be viewed in any generic unicode-based terminal. They only work in fonts with fixed letter width. Also has ascii-compatible character sets.

Docs: See PyPi.
Languages: Python 3
Version: 1.0.0
Download: PyPi

HashFish

The hashfish python module allows you to easily use salted HMACs for data hashing. It acts as a simple "wrapper" for some of the functions in the HMAC module from the standard library.

Docs: See PyPi.
Languages: Python 3
Version: 1.1.0
Download: PyPi

TimeQuery

TimeQuery is a Python module that allows you to get the time from anywhere in the world. It is designed to be more user-firendly then datetime and zoneinfo, and is programmed to act as a midway point between your application and these two modules. Designed to be as simple as possible, TimeQuery is small and yet effective in many ways.

Docs: See PyPi.
Languages: Python 3
Version: 1.0.0
Download: PyPi
